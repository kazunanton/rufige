УСТАНОВКА (5 МИНУТ)
────────────────────

1. Зарегистрируйтесь на хостинге и получите доступ к панели.

2. Загрузите два файла в корневую папку сайта (обычно public_html/
   или www/):
     • index.html
     • save.php

3. ВАЖНО: Измените ключ администратора в save.php!
   Откройте save.php и замените строку:
     define('ADMIN_KEY', 'hse2026secret');
   на свой секретный ключ, например:
     define('ADMIN_KEY', 'мой_уникальный_ключ_2026');

4. Убедитесь, что у папки сайта есть права на запись (обычно
   хостинг это делает автоматически). save.php сам создаст
   папку data/ при первом сохранении.

5. Откройте сайт в браузере — анкета готова!

ДОСТУП К ДАННЫМ
───────────────
  Админ-панель (#admin) — просмотр ответов, экспорт в JSON/CSV

  Прямые ссылки для скачивания (замените ВАШ_КЛЮЧ):
    Ответы:   https://ваш-домен.ru/save.php?action=export&key=ВАШ_КЛЮЧ
    Контакты: https://ваш-домен.ru/save.php?action=export_contacts&key=ВАШ_КЛЮЧ
    Статистика: https://ваш-домен.ru/save.php?action=stats&key=ВАШ_КЛЮЧ

КАК ЭТО РАБОТАЕТ
─────────────────
  • Респондент заполняет анкету → ответы отправляются на save.php
  • Ответы хранятся в файле data/responses.jsonl (одна строка = один респондент)
  • Контакты хранятся отдельно в data/contacts.jsonl
  • Папка data/ защищена от прямого доступа через .htaccess
  • Если сервер недоступен, ответы сохраняются в localStorage браузера

БЕЗОПАСНОСТЬ
────────────
  • Обязательно смените ADMIN_KEY на свой!
  • Папка data/ автоматически защищается файлом .htaccess
  • Данные доступны только по ключу администратора
  • CORS включён (анкета может быть встроена на другой домен)

БЕСПЛАТНЫЙ ВАРИАНТ
──────────────────
  Beget.com предлагает бесплатный хостинг с поддержкой PHP.
  Регистрация → загрузите два файла → готово.

ВОЗМОЖНЫЕ ПРОБЛЕМЫ
──────────────────
  «Ошибка 500» → проверьте права на папку (chmod 755)
  Ответы не сохраняются → убедитесь, что PHP включён на хостинге
  Не видно ответов в админке → проверьте, что ключ совпадает
